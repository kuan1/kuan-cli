# kuan-cli

> 一个简陋脚手架，个人练习项目，不建议其他人使用

# 使用

```bash
Usage: kuan-cli init [options] <app-name>

ejs模板快速创建项目

Options:
  -r, --repository <repository>  github/gitee仓库名称 (default: "zhongkuan/kuan-boilerplate")
  -b, --branch <branch-name>     仓库分支名称
  -d, --dir <dir>                二级目录
  -p, --platform <gitee|github>  gitee或者github (default: "gitee")
  -f, --force                    是否删除缓存，强制更新 (default: true)
  -h, --help                     display help for command
```
